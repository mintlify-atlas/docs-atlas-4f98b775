---
title: Delete a link
description: Delete a link for the authenticated workspace.
---

## Authentication

This endpoint requires authentication using a Bearer token.

```bash
Authorization: Bearer <token>
```

## Endpoint

<ParamField path="method" type="string" required>
  DELETE
</ParamField>

<ParamField path="endpoint" type="string" required>
  /links/:linkId
</ParamField>

## Path Parameters

<ParamField path="linkId" type="string" required>
  The id of the link to delete. You may use either `linkId` (obtained via `/links/info` endpoint) or `externalId` prefixed with `ext_`.
</ParamField>

## Response

<ResponseField name="id" type="string">
  The ID of the deleted link.
</ResponseField>

<RequestExample>

```bash cURL
curl -X DELETE "https://api.dub.co/links/clux0rgak00011..." \
  -H "Authorization: Bearer <token>"
```

```bash cURL (using externalId)
curl -X DELETE "https://api.dub.co/links/ext_123456" \
  -H "Authorization: Bearer <token>"
```

```javascript Node.js
const response = await fetch('https://api.dub.co/links/clux0rgak00011...', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer <token>'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.delete(
  'https://api.dub.co/links/clux0rgak00011...',
  headers={'Authorization': 'Bearer <token>'}
)

data = response.json()
```

</RequestExample>

```json Response

  "id": "clux0rgak00011..."
```
