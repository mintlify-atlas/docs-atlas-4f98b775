## Authentication

This endpoint requires authentication using a Bearer token.

<CodeGroup>
```bash cURL
curl -X POST https://api.dub.co/domains \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "slug": "acme.com",
    "expiredUrl": "https://acme.com/expired",
    "notFoundUrl": "https://acme.com/not-found",
    "placeholder": "https://dub.co/help/article/dub-links"
  }'
```

```typescript TypeScript
import { Dub } from 'dub';

const dub = new Dub({ token: process.env.DUB_API_KEY });

const domain = await dub.domains.create({
  slug: 'acme.com',
  expiredUrl: 'https://acme.com/expired',
  notFoundUrl: 'https://acme.com/not-found',
  placeholder: 'https://dub.co/help/article/dub-links',
});
```

```python Python
from dub import Dub

dub = Dub(token=os.environ['DUB_API_KEY'])

domain = dub.domains.create(
  slug='acme.com',
  expired_url='https://acme.com/expired',
  not_found_url='https://acme.com/not-found',
  placeholder='https://dub.co/help/article/dub-links',
)
```
</CodeGroup>

## Request Body

<ParamField body="slug" type="string" required>
  Name of the domain.
  
  Example: `acme.com`
</ParamField>

<ParamField body="expiredUrl" type="string">
  Redirect users to a specific URL when any link under this domain has expired.
  
  Example: `https://acme.com/expired`
</ParamField>

<ParamField body="notFoundUrl" type="string">
  Redirect users to a specific URL when a link under this domain doesn't exist.
  
  Example: `https://acme.com/not-found`
</ParamField>

<ParamField body="placeholder" type="string">
  Provide context to your teammates in the link creation modal by showing them an example of a link to be shortened.
  
  Example: `https://dub.co/help/article/dub-links`
  
  Maximum length: 100 characters
</ParamField>

<ParamField body="archived" type="boolean" default="false">
  Whether to archive this domain. `false` will unarchive a previously archived domain.
</ParamField>

<ParamField body="logo" type="string">
  The logo of the domain (uploaded image URL).
</ParamField>

<ParamField body="assetLinks" type="string">
  assetLinks.json configuration file (for deep link support on Android).
</ParamField>

<ParamField body="appleAppSiteAssociation" type="string">
  apple-app-site-association configuration file (for deep link support on iOS).
</ParamField>

## Response

<ResponseField name="id" type="string">
  The unique identifier of the domain.
</ResponseField>

<ResponseField name="slug" type="string">
  The domain name.
  
  Example: `acme.com`
</ResponseField>

<ResponseField name="verified" type="boolean">
  Whether the domain is verified.
</ResponseField>

<ResponseField name="primary" type="boolean">
  Whether the domain is the primary domain for the workspace.
</ResponseField>

<ResponseField name="archived" type="boolean">
  Whether the domain is archived.
</ResponseField>

<ResponseField name="placeholder" type="string | null">
  Provide context to your teammates in the link creation modal by showing them an example of a link to be shortened.
</ResponseField>

<ResponseField name="expiredUrl" type="string | null">
  The URL to redirect to when a link under this domain has expired.
</ResponseField>

<ResponseField name="notFoundUrl" type="string | null">
  The URL to redirect to when a link under this domain doesn't exist.
</ResponseField>

<ResponseField name="logo" type="string | null">
  The logo of the domain.
</ResponseField>

<ResponseField name="assetLinks" type="string | null">
  assetLinks.json configuration file (for deep link support on Android).
</ResponseField>

<ResponseField name="appleAppSiteAssociation" type="string | null">
  apple-app-site-association configuration file (for deep link support on iOS).
</ResponseField>

<ResponseField name="createdAt" type="date">
  The date the domain was created.
</ResponseField>

<ResponseField name="updatedAt" type="date">
  The date the domain was last updated.
</ResponseField>

<ResponseField name="registeredDomain" type="object | null">
  The registered domain record.
  
  <ResponseField name="id" type="string">
    The ID of the registered domain record.
  </ResponseField>
  
  <ResponseField name="autoRenewalDisabledAt" type="date | null">
    The date the domain auto-renew is disabled.
  </ResponseField>
  
  <ResponseField name="createdAt" type="date">
    The date the domain was created.
  </ResponseField>
  
  <ResponseField name="expiresAt" type="date">
    The date the domain expires.
  </ResponseField>
  
  <ResponseField name="renewalFee" type="number">
    The fee to renew the domain.
  </ResponseField>
</ResponseField>
