---
title: Retrieve a link
description: Retrieve the info for a link.
---

## Authentication

This endpoint requires authentication using a Bearer token.

```bash
Authorization: Bearer <token>
```

## Endpoint

<ParamField path="method" type="string" required>
  GET
</ParamField>

<ParamField path="endpoint" type="string" required>
  /links/info
</ParamField>

## Query Parameters

You must provide one of the following query parameters to identify the link:

<ParamField query="linkId" type="string">
  The unique ID of the short link.
  
  **Example:** `clux0rgak00011...`
</ParamField>

<ParamField query="externalId" type="string">
  This is the ID of the link in your database.
  
  **Example:** `123456`
</ParamField>

<ParamField query="domain" type="string">
  The domain of the link to retrieve. E.g. for `d.to/github`, the domain is `d.to`.
  
  **Note:** Must be used together with `key`.
</ParamField>

<ParamField query="key" type="string">
  The key of the link to retrieve. E.g. for `d.to/github`, the key is `github`.
  
  **Note:** Must be used together with `domain`.
</ParamField>

<ResponseField name="expiredUrl" type="string">

</ResponseField>

<ResponseField name="password" type="string">

</ResponseField>

<ResponseField name="proxy" type="boolean">

</ResponseField>

<ResponseField name="title" type="string">

<ResponseField name="description" type="string">
  The description of the short link.
</ResponseField>

<ResponseField name="image" type="string">
  The image of the short link.
</ResponseField>

<ResponseField name="rewrite" type="boolean">
  Whether the short link uses link cloaking.
</ResponseField>

<ResponseField name="doIndex" type="boolean">
  Whether to allow search engines to index the short link.
</ResponseField>

<ResponseField name="ios" type="string">
  The iOS destination URL for the short link.
</ResponseField>

<ResponseField name="android" type="string">
  The Android destination URL for the short link.
</ResponseField>

<ResponseField name="geo" type="object">
  Geo targeting information for the short link.
</ResponseField>

<ResponseField name="tags" type="array">
  The tags assigned to the short link.
</ResponseField>

  The unique ID of the folder assigned to the short link.

<ResponseField name="comments" type="string">
  The comments for the short link.

  The full URL of the short link, including the https protocol (e.g. `https://dub.sh/try`).
</ResponseField>

<ResponseField name="qrCode" type="string">
  The full URL of the QR code for the short link (e.g. `https://api.dub.co/qr?url=https://dub.sh/try`).
</ResponseField>

<ResponseField name="utm_source" type="string">
  The UTM source of the short link.
</ResponseField>

<ResponseField name="utm_medium" type="string">
  The UTM medium of the short link.
</ResponseField>

<ResponseField name="utm_campaign" type="string">
  The UTM campaign of the short link.
</ResponseField>

<ResponseField name="utm_term" type="string">
  The UTM term of the short link.
</ResponseField>

<ResponseField name="utm_content" type="string">
  The UTM content of the short link.
</ResponseField>

<ResponseField name="userId" type="string">
  The user ID of the creator of the short link.
</ResponseField>

<ResponseField name="workspaceId" type="string">
  The workspace ID of the short link.
</ResponseField>

<ResponseField name="clicks" type="number">
  The number of clicks on the short link.
</ResponseField>

<ResponseField name="leads" type="number">
  The number of leads the short link has generated.
</ResponseField>

<ResponseField name="conversions" type="number">
  The number of leads that converted to paying customers.
</ResponseField>

<ResponseField name="sales" type="number">
  The total number of sales (includes recurring sales) generated by the short link.
</ResponseField>

<ResponseField name="saleAmount" type="number">
  The total dollar value of sales (in cents) generated by the short link.
</ResponseField>

<ResponseField name="lastClicked" type="string">
  The date and time when the short link was last clicked.
</ResponseField>

<ResponseField name="createdAt" type="string">
  The date and time when the short link was created.
</ResponseField>

<ResponseField name="updatedAt" type="string">
  The date and time when the short link was last updated.
</ResponseField>

<RequestExample>

```bash cURL (by linkId)
curl -X GET "https://api.dub.co/links/info?linkId=clux0rgak00011..." \
  -H "Authorization: Bearer <token>"
```

```bash cURL (by domain + key)
curl -X GET "https://api.dub.co/links/info?domain=dub.sh&key=try" \
  -H "Authorization: Bearer <token>"
```

```bash cURL (by externalId)
curl -X GET "https://api.dub.co/links/info?externalId=123456" \
  -H "Authorization: Bearer <token>"
```

```javascript Node.js
const response = await fetch('https://api.dub.co/links/info?linkId=clux0rgak00011...', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer <token>'
  }
});

const data = await response.json();
```

```python Python
import requests

response = requests.get(
  'https://api.dub.co/links/info',
  headers={'Authorization': 'Bearer <token>'},
  params={'linkId': 'clux0rgak00011...'}
)

data = response.json()
```
