## What is Dub?

Dub is the modern, open-source link attribution platform trusted by world-class marketing teams from companies like Twilio, Buffer, Framer, Perplexity, Vercel, and Laravel.

Built for performance and scale, Dub powers **100M+ clicks** and **2M+ links** monthly, providing everything you need for:

- **Short links** - Create branded short links with custom domains
- **Conversion tracking** - Track clicks, leads, and sales with powerful analytics
- **Affiliate programs** - Run commission-based partner programs with built-in payouts

## Core Features

### Link Management

Create and manage short links at scale with advanced features:

- **Custom domains** - Use your own branded domains (e.g., `go.yourcompany.com`)
- **Custom slugs** - Choose memorable keys or auto-generate random ones
- **Link expiration** - Set expiration dates with custom redirect URLs
- **Password protection** - Secure links with password requirements
- **Device targeting** - Different destinations for iOS, Android, and geo-targeting
- **A/B testing** - Split traffic between multiple destination URLs

### Analytics & Attribution

Track every click with detailed analytics:

- **Real-time analytics** - Monitor clicks, leads, and conversions in real-time
- **Geographic data** - Track by country, region, city, and continent
- **Device insights** - Analyze by device type, browser, and operating system
- **Referrer tracking** - See where your traffic comes from
- **UTM parameters** - Full UTM parameter support and tracking
- **Conversion tracking** - Track leads and sales with revenue attribution

<Note>
  Analytics data is powered by Tinybird for blazing-fast queries on billions of events.
</Note>

### Conversion Tracking

Go beyond clicks with conversion tracking:

```typescript Example: Track a Lead
const response = await fetch('https://api.dub.co/track/lead', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer <your-api-key>',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    clickId: 'click_123abc',
    eventName: 'Sign Up',
    customerId: 'user_789xyz',
    customerEmail: 'user@example.com',
  }),
});
```

```typescript Example: Track a Sale
const response = await fetch('https://api.dub.co/track/sale', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer <your-api-key>',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    clickId: 'click_123abc',
    eventName: 'Purchase',
    customerId: 'user_789xyz',
    amount: 4999,
    currency: 'USD',
    metadata: {
      productId: 'prod_abc123',
      quantity: 1,
    },
  }),
});
```

### Affiliate Programs

Run commission-based partner programs with:

- **Partner management** - Invite and manage affiliate partners
- **Commission tracking** - Automatic commission calculations
- **Payout management** - Built-in payout processing via Stripe
- **Partner analytics** - Detailed performance dashboards for partners
- **Custom rewards** - Configure commission rates per partner group

## Tech Stack

Dub is built with modern, best-in-class technologies:

- **Next.js** - React framework for the web application
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first CSS framework
- **Prisma** - Next-generation ORM
- **Tinybird** - Real-time analytics engine
- **PlanetScale** - Serverless MySQL database
- **Upstash** - Redis for caching and rate limiting
- **Vercel** - Edge network for global deployment
