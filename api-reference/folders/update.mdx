## Authentication

This endpoint requires authentication using a Bearer token.

<CodeGroup>
```bash cURL
curl -X PATCH https://api.dub.co/folders/fold_abc123 \
  -H "Authorization: Bearer <token>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Q1 Marketing Campaigns",
    "description": "Updated description for Q1 campaigns"
  }'
```

```typescript TypeScript
import { Dub } from 'dub';

const dub = new Dub({ token: process.env.DUB_API_KEY });

const folder = await dub.folders.update('fold_abc123', {
  name: 'Q1 Marketing Campaigns',
  description: 'Updated description for Q1 campaigns',
});
```

```python Python
from dub import Dub

dub = Dub(token=os.environ['DUB_API_KEY'])

folder = dub.folders.update(
  'fold_abc123',
  name='Q1 Marketing Campaigns',
  description='Updated description for Q1 campaigns',
)
```
</CodeGroup>

## Path Parameters

<ParamField path="id" type="string" required>
  The ID of the folder to update.
</ParamField>

## Request Body

All fields are optional. Only include the fields you want to update.

<ParamField body="name" type="string">
  The name of the folder.
  
  Maximum length: 190 characters
</ParamField>

<ParamField body="description" type="string">
  The description of the folder.
  
  Maximum length: 500 characters
</ParamField>

<ParamField body="accessLevel" type="string">
  The access level of the folder within the workspace.
  
  Options: `write`, `read`, `none`
  
  Note: Changing access level requires Business plan or above.
</ParamField>

## Response

<ResponseField name="name" type="string">

<ResponseField name="description" type="string | null">
