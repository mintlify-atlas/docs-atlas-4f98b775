---
title: Bulk update links
description: Bulk update up to 100 links with the same data for the authenticated workspace.
---

## Authentication

This endpoint requires authentication using a Bearer token.

```bash
Authorization: Bearer <token>
```

## Endpoint

<ParamField path="method" type="string" required>
  PATCH
</ParamField>

<ParamField path="endpoint" type="string" required>
  /links/bulk
</ParamField>

## Request Body

<ParamField body="linkIds" type="array">
  The IDs of the links to update. Takes precedence over `externalIds`. Maximum of 100 IDs.
  
  **Example:** `["clux0rgak00011...", "clux0rgak00022..."]`
</ParamField>

<ParamField body="externalIds" type="array">
  The external IDs of the links to update as stored in your database. Maximum of 100 IDs.
  
  **Example:** `["123456", "789012"]`
</ParamField>

<ParamField body="data" type="object" required>
  The data to update on all specified links. All fields in the data object are optional.
</ParamField>

<Expandable title="data fields">

<ParamField body="data.url" type="string">
  The destination URL of the short link.
</ParamField>

<ParamField body="data.trackConversion" type="boolean">
  Whether to track conversions for the short link.
</ParamField>

<ParamField body="data.archived" type="boolean">
  Whether the short link is archived.
</ParamField>

<ParamField body="data.tagIds" type="array">
  The unique IDs of the tags assigned to the short link.
</ParamField>

<ParamField body="data.tagNames" type="array">
  The unique name of the tags assigned to the short link (case insensitive).
</ParamField>

<ParamField body="data.folderId" type="string">
  The unique ID existing folder to assign the short link to.
</ParamField>

<ParamField body="data.comments" type="string">
  The comments for the short link.
</ParamField>

<ParamField body="data.expiresAt" type="string">
  The date and time when the short link will expire at.
</ParamField>

<ParamField body="data.expiredUrl" type="string">
  The URL to redirect to when the short link has expired.
</ParamField>

<ParamField body="data.password" type="string">
  The password required to access the destination URL of the short link.
</ParamField>

<ParamField body="data.proxy" type="boolean">
  Whether the short link uses Custom Link Previews feature.
</ParamField>

<ParamField body="data.title" type="string">
  The custom link preview title (og:title).
</ParamField>

<ParamField body="data.description" type="string">
  The custom link preview description (og:description).
</ParamField>

<ParamField body="data.image" type="string">
  The custom link preview image (og:image).
</ParamField>

<ParamField body="data.video" type="string">
  The custom link preview video (og:video).
</ParamField>

<ParamField body="data.rewrite" type="boolean">
  Whether the short link uses link cloaking.
</ParamField>

<ParamField body="data.ios" type="string">
  The iOS destination URL for the short link.
</ParamField>

<ParamField body="data.android" type="string">
  The Android destination URL for the short link.
</ParamField>

<ParamField body="data.geo" type="object">
  Geo targeting information for the short link.
</ParamField>

<ParamField body="data.doIndex" type="boolean">
  Allow search engines to index your short link.
</ParamField>

<ParamField body="data.utm_source" type="string">
  The UTM source of the short link.
</ParamField>

<ParamField body="data.utm_medium" type="string">
  The UTM medium of the short link.
</ParamField>

<ParamField body="data.utm_campaign" type="string">
  The UTM campaign of the short link.
</ParamField>

<ParamField body="data.utm_term" type="string">
  The UTM term of the short link.
</ParamField>

<ParamField body="data.utm_content" type="string">
  The UTM content of the short link.
</ParamField>

<ParamField body="data.ref" type="string">
  The referral tag of the short link.
</ParamField>

<ParamField body="data.webhookIds" type="array">
  An array of webhook IDs to trigger when the link is clicked.
</ParamField>

</Expandable>

## Response

Returns an array of updated links or error objects.

<ResponseField name="[].id" type="string">
  The unique ID of the short link (success case).
</ResponseField>

<ResponseField name="[].domain" type="string">
  The domain of the short link (success case).
</ResponseField>

<ResponseField name="[].key" type="string">
  The short link slug (success case).
</ResponseField>

<ResponseField name="[].url" type="string">
  The destination URL of the short link (success case).
</ResponseField>

<ResponseField name="[].error" type="string">
  The error message (error case).
</ResponseField>

<ResponseField name="[].code" type="string">
  The error code (error case).
</ResponseField>

```bash cURL
  }'
```

```javascript Node.js
const response = await fetch('https://api.dub.co/links/bulk', {
  method: 'PATCH',
  headers: {

    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    linkIds: ['clux0rgak00011...', 'clux0rgak00022...'],

      archived: true,

    }
  })
});

const data = await response.json();
```

```python Python
  },
  json={
    'linkIds': ['clux0rgak00011...', 'clux0rgak00022...'],
    'data': {
      'archived': True,
      'tagIds': ['tag_123']
    }
  }
)

data = response.json()
```
