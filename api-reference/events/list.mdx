---
title: "List Events"
description: "Retrieve a paginated list of events for the authenticated workspace."
---

## Endpoint

<CodeGroup>
```bash GET
https://api.dub.co/events
```
</CodeGroup>

## Authentication

This endpoint requires authentication using a Bearer token.

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

<Note>
  This endpoint requires a Business plan or higher.
</Note>

## Query Parameters

### Event Configuration

<ParamField query="event" type="enum" default="clicks" required>
  The type of event to retrieve analytics for. Defaults to 'clicks'.
  
  Options: `clicks`, `leads`, `sales`
  
  <Expandable title="Example">
    ```
    event=leads
    ```
  </Expandable>
</ParamField>

### Pagination

<ParamField query="page" type="number" default={1}>
  The page number for pagination. Defaults to 1.
  
  <Expandable title="Example">
    ```
    page=2
    ```
  </Expandable>
</ParamField>

<ParamField query="limit" type="number" default={50}>
  The number of events to return per page. Max 1000 items per page.
  
  <Expandable title="Example">
    ```
    limit=100
    ```
  </Expandable>
</ParamField>

### Sorting

<ParamField query="sortBy" type="enum" default="timestamp">
  The field to sort the events by. The default is `timestamp`.
  
  Options: `timestamp`
  
  <Expandable title="Example">
    ```
    sortBy=timestamp
    ```
  </Expandable>
</ParamField>

<ParamField query="sortOrder" type="enum" default="desc">
  The sort order. The default is `desc`.
  
  Options: `asc`, `desc`
  
  <Expandable title="Example">
    ```
    sortOrder=asc
    ```
  </Expandable>
</ParamField>

### Link Identification

<ParamField query="domain" type="string">
  The domain to filter analytics for. Supports advanced filtering: single value, multiple values (comma-separated), or exclusion (prefix with `-`).
  
  <Expandable title="Examples">
    ```
    domain=dub.co
    domain=dub.co,google.com
    domain=-spam.com
    ```
  </Expandable>
</ParamField>

<ParamField query="key" type="string">
  The slug of the short link to retrieve analytics for. Must be used along with the corresponding `domain` of the short link.
  
  <Expandable title="Example">
    ```
    domain=dub.co&key=github
    ```
  </Expandable>
</ParamField>

<ParamField query="linkId" type="string">
  The unique ID of the link to retrieve analytics for. Supports advanced filtering: single value, multiple values (comma-separated), or exclusion (prefix with `-`).
  
  <Expandable title="Examples">
    ```
    linkId=link_123
    linkId=link_123,link_456
    linkId=-link_789
    ```
  </Expandable>
</ParamField>

<ParamField query="externalId" type="string">
  The ID of the link in your database. Must be prefixed with 'ext_' when passed as a query parameter.
  
  <Expandable title="Example">
    ```
    externalId=ext_123456
    ```
  </Expandable>
</ParamField>

### Time Range

<ParamField query="interval" type="enum">
  The interval to retrieve analytics for. If undefined, defaults to 24h.
  
  Options: `1h`, `24h`, `7d`, `30d`, `90d`, `ytd`, `1y`, `all`, `all_unfiltered`
  
  <Expandable title="Example">
    ```
    interval=7d
    ```
  </Expandable>
</ParamField>

<ParamField query="start" type="string">
  The start date and time when to retrieve analytics from. If set, takes precedence over `interval`. Format: ISO 8601 date string.
  
  <Expandable title="Example">
    ```
    start=2024-01-01T00:00:00Z
    ```
  </Expandable>
</ParamField>

<ParamField query="end" type="string">
  The end date and time when to retrieve analytics from. If not provided, defaults to the current date. If set along with `start`, takes precedence over `interval`.
  
  <Expandable title="Example">
    ```
    end=2024-12-31T23:59:59Z
    ```
  </Expandable>
</ParamField>

<ParamField query="timezone" type="string" default="UTC">
  The IANA time zone code for aligning timeseries granularity (e.g. America/New_York). Defaults to UTC.
  
  <Expandable title="Example">
    ```
    timezone=America/New_York
    ```
  </Expandable>
</ParamField>

### Other Filters

<ParamField query="url" type="string">
  The destination URL to retrieve analytics for. Supports advanced filtering.
  
  <Expandable title="Examples">

    ```
    url=https://example.com
    url=https://example.com,https://other.com
    url=-https://spam.com
    ```
  </Expandable>
</ParamField>

<ParamField query="root" type="boolean">
  Filter for root domains. If true, filter for domains only. If false, filter for links only. If undefined, return both.
</ParamField>

<ParamField query="saleType" type="enum">
  Filter sales by type: 'new' for first-time purchases, 'recurring' for repeat purchases. If undefined, returns both.
  
  Options: `new`, `recurring`
</ParamField>

<ParamField query="query" type="string">
  Search the events by a custom metadata value. Only available for lead and sale events. Max 10,000 characters.
  
  <Expandable title="Example">
    ```
    query=metadata['key']:'value'
    ```
  </Expandable>
</ParamField>

## Examples

<CodeGroup>
```bash cURL - List Click Events
curl -X GET "https://api.dub.co/events?event=clicks&limit=100&sortOrder=desc" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash cURL - List Lead Events
curl -X GET "https://api.dub.co/events?event=leads&start=2024-01-01T00:00:00Z&end=2024-01-31T23:59:59Z" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash cURL - List Sale Events for Link
curl -X GET "https://api.dub.co/events?event=sales&domain=dub.co&key=github&interval=30d" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```bash cURL - Paginated Events
curl -X GET "https://api.dub.co/events?event=clicks&page=2&limit=50" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```typescript TypeScript SDK - List Click Events
import { Dub } from "dub";

const dub = new Dub({
  token: "YOUR_API_KEY",
});

const events = await dub.events.list({
  event: "clicks",
  limit: 100,
  sortOrder: "desc",
});

console.log(events);
```

```typescript TypeScript SDK - List Lead Events
import { Dub } from "dub";

const dub = new Dub({
  token: "YOUR_API_KEY",
});

const events = await dub.events.list({
  event: "leads",
  start: new Date("2024-01-01"),
  end: new Date("2024-01-31"),
});

console.log(events);
```

```python Python SDK
from dub import Dub

dub = Dub(token="YOUR_API_KEY")

events = dub.events.list(
    event="sales",
    domain="dub.co",
    key="github",
    interval="30d"
)

print(events)
```

```go Go SDK
package main

import (
    "context"
    "fmt"
    "github.com/dubinc/dub-go"
)

func main() {
    client := dub.New(
        dub.WithSecurity("YOUR_API_KEY"),
    )

    events, err := client.Events.List(context.Background(), &dub.EventsListRequest{
        Event:   dub.String("clicks"),
        Limit:   dub.Int64(100),
        SortOrder: dub.String("desc"),
    })
    
    if err != nil {
        panic(err)
    }
    
    fmt.Println(events)
}
```
</CodeGroup>
